Full Stack Web Development TX00EY87-3001

Welcome!​

Full Stack Web Development is professionally rewarding. However, gaining the skills needed to enter the job pool may seem overwhelming. As it is, building modern web applications involves the integration of numerous technologies.

This course aims at making the learning process less intimidating.

Course Description​

This course introduces you to the skills and tools needed to increase your competitiveness as a web developer.

We will start with fundamental concepts of web development, covering briefly HTML, CSS, and the basics of JavaScript. As we progress we will discuss additional concepts and learn how to work with servers and databases.

In the process you will see how back end technologies connect to the front end to engineer a full stack web application. Also, you will be introduced to web technologies, frameworks, tools and libraries such as:

  • React
  • Node
  • Express
  • MongoDB
  • DevOps

By the end of the course, you will be equipped with enough knowledge to put together multiple technologies and build a single-page application with React and deploy it to cloud services.

Course Mechanics​

The first lecture will be on Thursday, 3rd of Mars from 17:00 -20:00.

Starting next week, the class will meet Tuesday and Thursday from 17:00 - 20:00 on Zoom. The link to the sessions is in OMA.

The course features lectures, assignments and a mini project. More information will be posted on the course web site. Students are expected to keep up to date with the announcements.

There is no required textbook for the course. In addition to the lectures, we will provide a set of reference material for additional information.

Learning Outcomes​

On completion of the course students will:

  • Gain experience with a set of modern development tools and techniques
  • Develop and deploy a fully functioning website
  • Understand React Hooks
  • Build a single-page application with React
  • Understand basic event handling in browser
  • Understand the role of web servers and server frameworks
  • Create custom web servers using Node.js and Express.js
  • Implement a RESTful backend API for storing and retrieving data
  • Authenticate, store, and structure user data

Contents​

Topics covered include:

  • Web developer tools
  • JavaScript best practices
  • Modular JavaScript development
  • Client-side app architecture
  • Server-side app architecture
